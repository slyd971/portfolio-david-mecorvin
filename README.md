# Portfolio CV

This is a portfolio and CV application built with Next.js. It showcases various projects and provides information about the developer's skills and experience.

## Features

- Built with Next.js using the App Router
- Styled with Tailwind CSS for a modern and responsive design
- ESLint enabled for maintaining code quality
- Organized structure with a `src` folder containing components and styles

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-cv
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000` to see the application in action.

## Directory Structure

- `src/app/page.js`: Main entry point for the application.
- `src/components/index.js`: Exports various React components.
- `src/styles/globals.css`: Global CSS styles including Tailwind CSS imports.
- `.eslintrc.json`: ESLint configuration file.
- `next.config.js`: Next.js configuration file.
- `package.json`: npm configuration file.
- `postcss.config.js`: PostCSS configuration file for processing CSS.
- `tailwind.config.js`: Tailwind CSS configuration file.
